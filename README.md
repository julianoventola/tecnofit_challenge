# Features

- Loading screen to check if user has already logged in
- Checking fields in login screen
- Keep user logged using shared preferences
- Logout button in home page

## Getting Started

- Clone the repository
- run in cmd:
  ```cmd
  flutter pub get
  flutter run
  ```
